# nlp-disaster-tweets
NLP Disaster Tweets Mini Project

Kaggle competition: [Natural Language Processing with Disaster Tweets](https://www.kaggle.com/c/nlp-getting-started/overview)

This Kaggle competition is about classifying texts. It is an excellent introduction to Natural Language Processing (NLP).

Requirements:
- Create Jupyter notebook with a description of the problem/data, exploratory data analysis (EDA) procedure, analysis (model building and training), result, and discussion/conclusion. 
- Generate a `submission.csv` file to submit to the kaggle competition.
